    Asymptotic Inference Analysis

    Python-based simulation study analyzing asymptotic inference in linear regression without assuming error normality.

    Key Results:

  • Data generation: 2000 samples were generated for each sample size
  • Test size: As sample size increased, the test size approached that of the Normal distribution
  • Fastest Convergence: Beta distribution
  • Slower Convergence: t-Student and Uniform distributions
  • Potential Applications: evaluation of public policy impacts, modeling of wages, model energy demand

    Finite Sample Properties of the FGLS Estimator

    An analysis of the finite sample behavior of the Feasible Generalized Least Squares (FGLS) estimator.

    Key Results:

  • Test Size: For N > 200, the test size converges to the expected 1% and 5% significance levels
  • Test Power: For small sample sizes, the power of the test for H₀: β₁ = 0 and H₀: β₁ = 0.4 converges to 1
  • Estimation: The coefficients estimated by GLS show smaller Standard Deviations compared to those estimated by FGLS
